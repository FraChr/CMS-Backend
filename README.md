# Backend CMS API

## Overview
This is an api built with ef core minimal apis.
it uses Sqlite database with persistant local storage for upload and migrations via ef core


## Tech Stack
- .NET 10
- ASP.NET Core Web Api
- Entity Framework Core
- SQLite


## Prerquisite
- .NET SDK 10

## Frontend

The frontend for this project is located at a seperate repo

[Vue Frontend](https://github.com/FraChr/CMS-Frontend)

## Run

- Clone repo

```sh
cd CMS-Backend/Api
```
#### Run API
```sh
dotnet run
```
